# AR Otohazama

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

AR Otohazama is a web-based AR application that displays a floating 3D scene.

## Demo
The demo is available at https://code4fukui.github.io/ar-otohazama/

## Features
- Renders a 3D scene with floating cubes
- Uses the AR Globe and egxr.js libraries

## Requirements
This project requires a modern web browser with WebGL and WebXR support.

## Usage
To run the application locally, simply open the `index.html` file in a web browser.
