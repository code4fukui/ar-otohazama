# AR Otohazama

AR Otohazamaは、WebブラウザのAR機能を使って3D空間で音を視覚化するWebアプリケーションです。様々な音楽を立体的に体験できます。

## デモ
デモは以下のURLで確認できます: https://code4fukui.github.io/ar-otohazama/

## 機能
- 3D空間にオブジェクトを表示し、音の高さに合わせて動きを再現
- WebXR APIを使ったAR体験

## 必要環境
- Webブラウザがバーチャルリアリティ (WebXR) 機能をサポートしている必要があります
- スマートフォンやタブレットなどのモバイル端末が推奨されます

## 使い方
1. Webブラウザでデモページを開きます
2. 端末のカメラ使用を許可すると、ARモードが開始されます
3. 3D空間に音の形状が表示され、音の変化に合わせて動きます
