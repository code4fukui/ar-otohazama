# AR Otohazama

AR Otohazamaは、浮遊する3Dシーンを表示するウェブベースのARアプリケーションです。

## デモ
デモは https://code4fukui.github.io/ar-otohazama/ でご覧いただけます。

## 機能
- 浮遊するキューブの3Dシーンをレンダリング
- AR Globeおよびegxr.jsライブラリを使用

## 必要条件
本プロジェクトを実行するには、WebGLおよびWebXRをサポートするモダンウェブブラウザが必要です。

## 使い方
アプリケーションをローカルで実行するには、ウェブブラウザで`index.html`ファイルを開いてください。
